# MediaMTX Matrix Pro v3.4.0

A professional, high-contrast monitoring dashboard designed for **MediaMTX** and **Frigate NVR**. Built with Next.js 15, Tailwind CSS, and optimized for low-latency video management.

### Key Features:
* **High-Contrast Comfort:** Midnight Slate & Silver theme based on GitHub Primer.
* **Stream Matrix:** Instant routing of publishers to Frigate recording slots.
* **Protocol Hub:** Quick access to SRT, RTMP, and RTSP publishing URLs.
* **Production Ready:** Optimized Docker multi-stage builds with Nginx reverse proxy.
* **Multi-Node:** Decoupled architecture allowing the dashboard to run across firewalls.

### Tech Stack:
* **Framework:** Next.js 15 (App Router)
* **Styling:** Tailwind CSS (Comfort Slate Palette)
* **Icons:** Lucide React
* **Deployment:** Docker / Docker Compose / Nginx
